# Interactivity Example

Example includes: *interactivity*, *components*, *threads*, *stages*, owner-scoped button panels, and a richer multi-step task workflow that demonstrates `WithV2Components()`, branching buttons, modal capture, and follow-up handling with Components V2.

To get the bot up and running, run the following command:

```
dotnet run <someBotTokenHere>
```

You can also supply the token through the `DISCATSHARP_TOKEN` or `DISCORD_TOKEN` environment variable.
